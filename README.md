# JFODACalculator

This is a calculator for our 3rd year software engineering project.
JFODA stands for Jansen, Forde, O'Dwyer, Aylward.

This calculator is split into 4 distinct components:
1. Tokenizer
2. Infix To Posfixer
3. Code Generator
4. Virtual Machine

## Compiling

To compile simply run
```make```
